
1. Создайте файл index.html

2. С помощью "!" создайте структуру страницы

4. В теге <head> откройте тег <style> напишите все необходимые CSS свойства для того чтобы создать:

  	1. квадрат зеленого цвета размером 100px * 100px с помощью тега <div>
	2. круг красного цвета также размером 100px * 100px с помощью тега <span>
	3. выстройте их в одну линию с отступом от квадрата в 40px.

Примечание: (используйте CSS свойства: width, height, background-color, border-radius, display, margin.
	Чтобы создать круг необходимо задать ширину, высоту и придать ему округлую форму, используйте свойство border-radius: 50%;
	для того, чтобы к строчному элементу применились ранее указанные свойства, используйте свойство display с нужным значением!
	для того чтобы выстроить их в один ряд, назначьте блочному элементу необходимое значение display и сделайте отступ от него 
	с помощью margin)
